### Installation

1. Clone the repo
   ```sh
   $ git clone https://github.com/Szezi/ToToList_Tutorial
   ```
2. Install packages
   ```sh
   $ pip install -r requirements.txt
   ```
3. Run
   ```sh
   $ python manage.py runserver 
   ```
